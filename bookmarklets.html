Bookmarklets

Bookmarklets för användning med Google Chrome.



SMH GetObject JSON

Se ett objekts JSON data som kommer från Quedro.

  • Dra denna knapp/länk till bokmärkesfältet i Chrome:
    SMH JSON
  • Gå till en objektsida.
  • Klicka på bookmarklet:en som du nu har i bokmärkesfältet.
  • Du skickas till en onlinetjänst som visar upp JSON data för objektet.
  • Du kan växla till Tree View Mode
  • Du kan söka.

SMH växla ett objekt mellan olika miljöer

Växla visning av ett objekt mellan Prod - Stage, Prod - Test, Prod - Local.

  • Dra dessa knappar/länkar till bokmärkesfältet i Chrome:
    Toggle Prod > Stage Toggle Prod > Test Toggle Prod > Local
  • Gå till en objektsida.
  • Klicka på bookmarklet:en som du nu har i bokmärkesfältet.
  • Samma objekt visas i den miljön som du valde.

SMH Reco uppgifter

Hämta en medarbetares Reco-uppgifter (venueId och employeeId).

  • Dra denna knapp/länk till bokmärkesfältet i Chrome:
    Reco ID
  • Gå till en mäklares Reco-sida (här finns alla SMH:s kontor listade, klicka på ett kontor för att se medarbetare som tillhör kontoret).
  • Klicka på bookmarklet:en som du nu har i bokmärkesfältet.
  • En alert visas med Reco-uppgifter (samma information skrivs också ut i console).
  • Du kan kopiera CTRL + C informationen i alert:en.

Vad är en bookmarklet?

"A bookmarklet is a bookmark stored in a web browser that contains JavaScript commands that add new features to the browser. Bookmarklets are unobtrusive JavaScripts stored as the URL of a bookmark in a web browser or as a hyperlink on a web page. Bookmarklets are usually JavaScript programs." Wikipedia
